1. Financial Spreading as a Service (FSaaS) : Amazon Web Services

Description:
● Service that provides spreading of Financial Documents (Balance, Income and Cashflow Sheets) in a structured format.
● Document is uploaded in Amazon S3 bucket which acts a trigger to an AWS Lambda function. The document then undergoes OCR using Amazon Textract and all the text data is stored in another S3 bucket.
● The S3 bucket then triggers another Lambda function responsible for capturing the relevant Financial pages using ML and tables are extracted from the text data in a structured JSON format.
Tech stack: Amazon Textract, Amazon Sagemaker, AWS SQS and SNS,AWS Lambda, S3, API Gateway, Python

2. Remax : Amazon Web Services

Description:
● Contactless delivery center created using Amazon Connect to re-direct call flows and answer user queries without human interference.
● Customer calls are initially redirected to the Amazon Connect Service. Connect is integrated with Amazon Lex in the back end to match the user query and answers questions.
● If the Lex Chatbot is unable to fulfill customers' needs, calls are redirected to the agents through Connect.
● Amazon Connect also provides real time call data analysis as well as complete call reports.
Tech stack: Amazon Connect, Amazon Lex, AWS Firehose, AWS Kinesis, AWS Lambda, S3, API Gateway, Python
